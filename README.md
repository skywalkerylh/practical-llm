## How to use the model?

| Method |  |
|--------|--------|
| Chat Interface | 透過聊天介面使用 |
| Cloud APIs | • Pay per API<br>• Langchain 作為中間層整合不同模型 API 和服務，提供統一介面和常見應用服務<br>• Managed AI cloud service (如 Azure ML, Google Vertex) 提供模型訓練與部署<br>• 可透過 Langchain 直接呼叫 Azure 模型，簡化 API 使用 |
| Direct Interface | 從 Hugging Face 下載模型參數自行使用 |
